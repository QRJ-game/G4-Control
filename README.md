# G4 Control v0.9.2 Public Beta

Расширенный локальный WebUI для мобильного роутера G4 PRO.

Это не полная прошивка и не образ разделов. Установщик сохраняет заводской интерфейс, внедряет G4 Control в штатный `/index.html` через bind-mount и добавляет небольшой локальный ARM helper для функций, которых нет в обычном WebUI.

**Авторы:** sleo441 и ChatGPT.
